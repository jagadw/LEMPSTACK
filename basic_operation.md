# Update : Now, we can start or stop nginx & mysql using script.
- Start via bash
```bash
sudo systemctl start nginx
```
- Start via bash
```bash
sudo systemctl stop nginx
```

# NGINX
- Start:
```bash
sudo systemctl start nginx
```
- Reload:
```bash
sudo nginx -S reload
```
- Stop:
```bash
sudo systemctl stop nginx
```
# MYSQL
- Start:
```bash
sudo systemctl start mysql
```
- Stop:
```bash
sudo systemctl stop mysql
```
